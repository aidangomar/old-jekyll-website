App Creators (suggested)

GameSalad - NO CODING REQUIRED

Free for Mac. Allows you to build games of all sorts using their friendly drag-and-drop environment, and can export native apps for HTML5 ( web ), iOS, Mac, Android, etc. http://www.gamesalad.com/

PhoneGap - WEB CODING REQUIRED

Free for Any Operating System. This entirely web-based system turns your HTML5, JS, CSS3 source code into a native app for almost any mobile platform, including: iOS, Android, Blackberry, webOS, Symbian, etc. http://www.phonegap.com/

Browsers (suggested)

Google Chrome - WEBKIT BROWSER

Great Webkit Browser. Good for both Developer use, and regular use. Supports almost all new web features including CSS3 and HTML5. http://www.google.com/chrome/

Xcode - OBJECTIVE-C CODING REQUIRED

The original iOS App Developer Kit. Allows you to simulate and build native iPhone apps using Objective-C coding. If you are more comfortable with web programming and don’t have a big understanding of Objective-C, I would suggest using PhoneGap.

http://developer.apple.com/

Flexiwatch

A flexible, gesture-based stopwatch written for the web. Because of this, it runs on almost any device, and is even optimized for all mobile devices with a browser.

Blend: Fruity Insanity!

Easy to pick up but challenging to master, Blend is a game of reflexes and intuition. You'll be having fun seconds after opening the game! Blend is a fast-paced matching game, built during the 2013 MakeGamesWithUs Internship. Optimized for iPhones, iPods, and iPads. Works on most Android devices as well.

MITTAL.NET

A revamp of my dad's old website, mittal.net. Built to try and make the site faster and cleaner, as well bring much of the code up to modern web standards.

jQRy

A simple JavaScript-based microlibrary to easily generate 2D QR codes, 1D barcodes, and data matrices. Built in an attempt to simplify generating scannable codes in JavaScript.

MathNinja

Simple math game, throwing a problem at the user, and depending on whether the user got it correct or not, promoted the user one 'belt level'. Became the first app I ever released on the market. It was released on the Google Chrome Web Store and the Apple App Store. One of the first cross-platform apps I published. Created when I was 11 years old.
